Typography tokens While earlier iterations of Material relied on hard-coded style values for typography, design tokens offer an improved way to define typography in products. Tokens assign an element's type style by a configurable value, rather than a set value. Type roles describe size—such as small, medium, and large—enabling them to adapt and respond to the device or context.
